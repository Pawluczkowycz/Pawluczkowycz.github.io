Travel Agency Website Mockup

World Traveler is a fictitious travel agency site. My design employs a flat design, using simple photography from unsplash, solid colors and white space for each section.

The site uses the the lightweight Skeleton CSS framework, and includes David Walsh's debounce function for scrolling animation, Google Fonts, a simple homemade jQuery lightbox, and media queries for a responsive layout.
