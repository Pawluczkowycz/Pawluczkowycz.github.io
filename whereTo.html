Where To? Web App

This web app is based on a project from a Udacity project. It uses javascript to take street and city form inputs to make a AJAX call to Google Streetview API to retrieve an image of the street location. AJAX is used to connect with the New York Times API to retrieve the 10 lastest New York Times articles about the city. And a AJAX call is also used to connect to Wikipedia and receive JSON data for relevant articles about the city.
