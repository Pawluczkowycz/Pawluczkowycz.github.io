Weather Web App

This weather app uses AJAX to call the Weather Underground API to retrieve the current 3 and 7 day weather forecast. A location is first attempted using the browser GeoLocation API, if denied text input is used. I have also strutured the main JavaScript file to utilize a MVC pattern. The app was kept super-simple using a responsive, flexible grid built using percentages and media queries, a Google Web font and a little jQuery amd Velocity.js.
