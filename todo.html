jQuery Mobile ToDo App

This web app is based on a project from Udemy course. It uses javascript and jQuery Mobile to store a to-do list using the localStorage web API. I have deviated from the course by refactoring the javascript file to utilize an object literal module structure. Some additional CSS styling was also included.
